
# Shoot the Goblin

<br>
A basic Python Application made using PyGame.

<br>
Requirement:<br>
<ul>
    <li>Python: 3.8.8</li>
    <li>Pygame: 2.1.2</li>
</ul>
To Install Pygame:<br>
```
pip install pygame
```
